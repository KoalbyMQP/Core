# zaraos_interfaces

Shared ROS2 message and service definitions used by all ZaraOS nodes. This is a CMake package that generates C++ and Python type bindings via `rosidl`.

## Messages

| Message | Description |
|---------|-------------|
| `AuthCode` | Pairing code generated by Cortex (6-digit code, expiry, timestamp) |
| `AuthStatus` | Pairing outcome: `paired`, `expired`, or `failed` |
| `InstanceState` | Container state change: `starting`, `running`, `stopping`, `stopped`, `crashed` |
| `LogLine` | Single container log line (instance ID, stream, content, sequence number) |
| `SystemError` | System-level error/warning for UI display (source, severity, message) |

## Services

| Service | Description |
|---------|-------------|
| `WifiScan` | Request a WiFi network scan. Returns SSIDs with signal strengths. |
| `WifiConnect` | Connect to a WiFi network with SSID + password. Returns success/message. |

## Dependencies

- `std_msgs`
- `builtin_interfaces`

## Build

```bash
colcon build --packages-select zaraos_interfaces
```

This package must be built before any package that imports these types (`cortex_bridge`, `face_ui`, `startup_wizard`). Source the workspace after building:

```bash
source install/setup.bash
```
